## Features
- Facebook  Log In
- Google Sign In
- Email/Pass Registration / Log In
- Photo Messages
- Video Messages
- Real Time Conversations
- Location Messages
- Search for Users
- Deleting Conversations
- User Profile
- Dark Mode Support

From iosacademy.io

## Set Up
Don't forget to run 'pod install' after cloning the repo, to install cocoapod dependencies.
